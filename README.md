# Task-1-and-Task-2-
Task 1 – Basic Mathematical Operations
Its a Python program that asks the user to enter two numbers. It perform four operations on those numbers: addition, subtraction, multiplication, and division. It display the result of each operation on the screen.
Task 2 – Personalized Greeting
Its a Python program that takes the user’s first name and last name as input. Then combine them to create the full name. It print a greeting message using that full name.
